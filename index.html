Helper 소개

기술로 일상을 더 편리하게

Helper는 AI 기반 서비스와 모바일 개발 전문 회사입니다. 특히 접근성 향상을 통해 모든 사용자가 기술의 혜택을 누릴 수 있도록 하는 것이 우리의 목표입니다.

대표자: 김성준

iOS 전문 개발자로서 사용자 중심의 혁신적인 모바일 솔루션을 개발하고 있습니다. 특히 AI 기술을 활용한 접근성 서비스 개발에 열정을 가지고 있으며, 기술을 통해 더 나은 세상을 만들어가는 것이 목표입니다.

핵심 가치

🎯 사용자 중심 설계
🔧 기술 혁신
♿ 접근성 향상
🚀 지속적인 개선

주요 서비스

📱

iOS 앱 개발

Swift, SwiftUI, UIKit을 활용한 네이티브 iOS 애플리케이션 개발. 사용자 경험을 최우선으로 하는 직관적이고 성능 최적화된 앱을 제작합니다.

🤖

AI 통합 솔루션

인공지능 기술을 모바일 애플리케이션에 통합하여 스마트한 사용자 경험을 제공합니다. 음성 인식, 이미지 처리 등 다양한 AI 기능을 구현합니다.

♿

접근성 서비스

시각장애인과 같은 특수 사용자를 위한 접근성 향상 서비스를 개발합니다. 모든 사용자가 기술의 혜택을 누릴 수 있도록 지원합니다.

주요 프로젝트

🔍 LuminaView

시각장애인을 위한 AI 기반 음성 안내 서비스입니다. 실시간 객체 인식과 텍스트 인식 기능을 통해 시각장애인의 일상생활을 돕는 혁신적인 접근성 솔루션입니다.

Swift Core ML Vision Framework AVFoundation

💬 오테톡 (Ototalk)

실제 외주 프로젝트로 진행된 커뮤니케이션 앱입니다. 사용자 친화적인 인터페이스와 실시간 메시징 기능을 구현하여 원활한 소통을 지원하는 모바일 메신저 애플리케이션입니다.

Swift UIKit Firebase Real-time DB

💼 무선/유선 과금 시스템

세종텔레콤에서 개발한 통신 과금 처리 시스템입니다. 고객 정보 조회 성능을 개선하고 이름 변경 처리 자동화를 구현하여 업무 효율성을 크게 향상시켰습니다.

Java Spring Oracle RESTful API
